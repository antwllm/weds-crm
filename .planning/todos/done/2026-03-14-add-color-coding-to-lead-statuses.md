## Problem

Les statuts des leads sont affiches sans distinction visuelle, ce qui rend difficile de reperer rapidement l'etat d'un lead dans la liste.

## Solution

Ajouter un code couleur aux badges/labels de statut dans le frontend (ex: vert pour gagne, rouge pour perdu, orange pour en cours, etc.).
