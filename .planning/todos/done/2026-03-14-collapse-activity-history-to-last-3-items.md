## Problem

L'historique d'activite affiche toutes les actions, ce qui prend trop de place et alourdit l'interface, surtout sur les leads avec beaucoup d'historique.

## Solution

Afficher uniquement les 3 dernieres actions par defaut, avec un bouton "Voir plus" pour derouler l'historique complet.
