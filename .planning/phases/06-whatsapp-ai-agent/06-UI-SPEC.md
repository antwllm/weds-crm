## Spacing Scale

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (regular) | 1.5 |
| Label | 12px | 500 (medium) | 1.4 |
| Heading | 20px | 700 (bold) | 1.2 |

## Layout Specifications

### AI Agent Banner (in WhatsAppChat)

```
+------------------------------------------------------------------+
| [Switch] Agent IA                        [Badge: En attente...]  |
+------------------------------------------------------------------+
| Chat messages area (existing)                                     |
```

- Position: Between the tab navigation and the chat message area
- Height: auto (single row, vertically centered content)
- Padding: `px-3 py-2` (12px horizontal, 8px vertical)
- Background: `bg-muted/50` (subtle differentiation from chat area)
- Border: `border-b` bottom border only
- Layout: `flex items-center justify-between`
- Left side: Switch + label "Agent IA" with `gap-2`
- Right side: Handoff badge (conditionally rendered)

### ChatBubble AI Badge

```
+--------------------------------------+
| Vous              [Badge: IA]        |
| Message text here                    |
|                         il y a 2min  |
+--------------------------------------+
```

- Position: Inline next to the sender label ("Vous")
- Badge size: `h-4 px-1.5 text-[10px]` (smaller than default badge)
- Badge content: "IA" (two letters only)
- Badge color: `bg-violet-100 text-violet-700` (purple semantic)
- Only shown when `message.sentBy === 'ai'`

### Disable AI Confirmation Dialog

- Size: `sm` (compact confirmation)
- No media icon
- Standard AlertDialog layout: header (title + description) + footer (cancel + action)

### WhatsAppAgentSettings (Settings Page)

```
+------------------------------------------------------------------+
| Prompt systeme WhatsApp                                           |
| +--------------------------------------------------------------+ |
| | [Textarea: 10 rows, font-mono text-sm]                       | |
```

## Interaction States

### AI Toggle Switch

| State | Visual |
|-------|--------|
| Off (default) | Switch unchecked, muted track color |
| On | Switch checked, primary track color |
| Toggling on | Immediate switch, optimistic update |
| Toggling off | AlertDialog opens first, switch stays on until confirmed |
| Loading | Switch disabled with `opacity-50` |
| Error | Toast with error message, revert switch state |

### Handoff Badge

| State | Visual |
|-------|--------|
| No handoff | Badge not rendered |
| Active handoff (`whatsappAiHandoffAt` is set and no human reply since) | Orange badge visible: "En attente de reponse humaine" |
| Resolved (human has replied) | Badge not rendered |

### Settings Form

| State | Visual |
|-------|--------|
| Loading | "Chargement..." centered text (existing pattern) |
| Clean (no changes) | Save button disabled |
| Dirty (unsaved changes) | Save button enabled |
| Saving | Save button shows "Sauvegarde...", disabled |
| Saved | Toast notification, button reverts to disabled |
| Error | Toast with error, button re-enabled |

## Accessibility Notes

- AI toggle Switch: must have accessible label "Agent IA" via `aria-label` or associated `<label>`
- AI Badge on messages: purely decorative indicator, no interactive role needed
- Handoff badge: `role="status"` for screen reader announcement
- AlertDialog: inherits keyboard navigation from base-ui (Escape to close, Tab between actions)
- Settings textareas: associated labels via `htmlFor`/`id` pairing
