   - sendWhatsAppMessage(phoneNumberId: string, accessToken: string, to: string, body: string): Promise<string>
     Calls axios.post to `https://graph.facebook.com/v21.0/${phoneNumberId}/messages` with messaging_product='whatsapp', recipient_type='individual', to, type='text', text: { body }. Returns response.data.messages[0].id.
