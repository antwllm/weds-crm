    1. Create client/src/hooks/useLeads.ts with TanStack Query hooks:
       - useLeads(filters?: LeadFilters): useQuery that fetches GET /api/leads with query params built from filters. Query key: ['leads', filters].
       - useCreateLead(): useMutation for POST /api/leads. On success: invalidate ['leads'].
       - useUpdateLead(): useMutation for PATCH /api/leads/:id. On success: invalidate ['leads'].
       - useUpdateLeadStatus(): useMutation for PATCH /api/leads/:id with {status}. Implements OPTIMISTIC UPDATE:
         * onMutate: cancelQueries(['leads']), snapshot previous, update cache by mapping lead status
         * onError: rollback to snapshot, show toast.error("Erreur lors du changement de statut")
         * onSettled: invalidateQueries(['leads'])
       - useDeleteLead(): useMutation for DELETE /api/leads/:id. On success: invalidate ['leads'].
